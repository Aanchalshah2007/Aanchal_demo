Jarvis AI Voice Assistant
A Python-based AI voice assistant that listens to voice commands, responds intelligently using Google Gemini AI, and performs tasks like opening websites, playing music, and reading news.

Features
Wake-word detection ("Jarvis")
Speech recognition
Text-to-speech responses
AI-powered conversation
Music playback
Live news headlines
Tech Stack
Python
SpeechRecognition
pyttsx4
Google Gemini API
NewsAPI
How to Run
Clone the repository
Install dependencies
pip install -r requirements.txt
Set environment variables
Run
python main.py
Security
All API keys are managed using environment variables and are excluded from version control.
